Background

Håstad's broadcast attack (1985) exploits RSA with small e when the same message is sent to multiple recipients.


With e = 3 recipients, the Chinese Remainder Theorem reconstructs m³ over the integers. Since m < min(nᵢ), we have m³ < n₁n₂n₃, so taking the cube root gives m exactly — no private key needed.

⚠ KEY INSIGHT m < nᵢ → m³ < n₁·n₂·n₃ → CRT gives m³ over ℤ (not mod anything!). The cube root is then trivial.

CRT Formula

N = n₁ × n₂ × n₃
Nᵢ = N / nᵢ
Mᵢ = Nᵢ⁻¹ mod nᵢ
m³ = (Σ cᵢ·Nᵢ·Mᵢ) mod N
Since m³ < N, no modular reduction occurs

# Step 1: CRT
N = n1*n2*n3
N1,N2,N3 = N//n1, N//n2, N//n3
M1 = pow(N1,-1,n1); M2 = pow(N2,-1,n2); M3 = pow(N3,-1,n3)
m_cubed = (c1*N1*M1 + c2*N2*M2 + c3*N3*M3) % N

# Step 2: Integer cube root (Newton's method)
x = m_cubed
for _ in range(200):
    x = (2*x + m_cubed // (x*x)) // 3
    if x*x*x == m_cubed: break

# Step 3: Decode
m = x.to_bytes((x.bit_length()+7)//8, 'big')
print(f"FlagVault{{{m.decode()}}}")
